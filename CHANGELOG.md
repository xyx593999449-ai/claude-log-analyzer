# Changelog

## [Unreleased]

### Added

- 新增首页日志导入折叠区，支持核实日志和质检日志分阶段上传。
- 新增核实 / 质检执行概览横向对比卡片，并补充平均成本指标与悬浮说明。
- 新增首页概览接口中的全量流程状态统计，用于顶部流程看板展示。
- 新增首页概览中的“质检不通过”统计，用于顶部流程看板的异常信息展示。
- 新增任务流程卡片中的共享展开交互，核实模块与质检模块可联动展开。
- 新增任务流程卡片中的核实摘要与质检摘要展示，收起状态下也可直接查看。
- 新增任务流程卡片的证据摘要展示，按 `poi_verified.evidence_record` 聚合证据总量与来源数量标签。
- 恢复任务流程卡片中的核实模块与质检模块详情，并展示执行状态、时间、Token、成本与错误摘要。
- 新增日志详情页阶段切换视图与异常告警展示。
- 新增接近 IDE 风格的滚动条样式，用于时间线、工具分析和原始日志区域。

### Changed

- 重构首页视觉层级，弱化说明性文案，保留核心业务信息展示。
- 将首页当前页流程分布从柱状图改为任务流转状态板，直接展示各状态任务量与占比。
- 将首页右侧异常概览改为异常标签聚合展示。
- 将首页顶部主看板改为全量流程执行状态展示，并将核实自动化率、核实质量、异常任务下移到流程看板下方。
- 将首页顶部主看板从漏斗流方案调整为地铁线路式流程带方案，强化主流程平铺展示和可读性。
- 将任务列表顶部的流程分布与异常标记改为纵向排列，优先保障流程节点平铺展示效果。
- 移除首页中的“核实状态分布”和“重点指标”两个独立总览模块，避免与下方执行概览和流程分布重复。
- 调整任务详情卡片结构：流程轨道移入默认收起区，地址前置到任务标题下方，元数据拆分回核实模块与质检模块。
- 调整任务详情列表布局，默认仅展示上半区摘要，展开后再显示下半区内容。
- 将任务元信息调整到异常区域之前展示。
- 将查看日志详情入口改为独立按钮。
- 优化日志详情页整体样式，使其与主看板保持一致。

### Fixed

- 修复平均成本卡片缺失问题。
- 修复多处前端中文文案乱码问题。
- 清理页面中类似需求备注的展示文案，避免界面出现说明稿式文字。
- 修正对“删除两个模块”需求的误解，恢复核实详情与质检详情可见性。
- 修复首页全量流程看板中的装饰虚线穿过节点卡片、影响展示的问题。
- 修复文件上传导入后任务日志详情无法展示原始日志的问题，并改为按任务所属导入批次读取日志而非全局最近一次导入。
