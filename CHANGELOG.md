# Changelog

本文件记录项目关键变更。

## [Unreleased]

### Added
- 新增大 POI 看板首页（概览卡、任务列表、分页筛选）。
- 新增核实/质检双窗口日志上传，支持批量选择、拖拽上传、自动识别与手动修正。
- 新增“自动化率”指标（核实概览）。
- 新增“核实质量”指标（质检概览）。
- 新增总成本展示（概览）。

### Changed
- 日志详情页改为复用旧风格分析能力：Timeline / ToolAnalysis / TokenAnalysis / RawLog。
- 成本口径统一为 Token 计算（GLM 输入 4 元/百万，输出 18 元/百万）。
- 指标展示层级优化：关键比率与任务量同级突出展示。
- 首页视觉重绘：上传区、概览卡、筛选与表格增强层次与可读性。

### Fixed
- 修复后端 SQL 乱码导致的概览查询失败问题。
- 修复 UTF-8/BOM 读取与导入稳定性问题。
- 修复前后端字段对齐问题（总 Token、总成本等）。

### Docs
- 重写项目级 `README.md`，补充 SOP、流程图、节点说明与指标口径。
- 更新项目级 `CHANGELOG.md`，同步当前迭代变更。
